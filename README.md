# Finite Heap #

Finiteheap is my person blog/site built with
[Jekyll](http://jekyllrb.com).

## Libraries Used ##
+ [Jekyll](http://jekyllrb.com)

## Dev Tools Used ##
+ [Capistrano](http://capistranorb.com/)
+ [Emacs](http://emacsformacosx.com)

## Workflow ##

```bash
$ jekyll build
# Stage, commit and push site changes to Github
$ cap deploy
```
